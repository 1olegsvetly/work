3. Функциональные требования (ядро — сохраняется из v1.0)
3.1. Базовая логика (без изменений)
Загрузка списков прокси из конфигурируемых источников
3 случайных прокси из каждого источника
Проверка работоспособности (HTTP/HTTPS/SOCKS4/SOCKS5)
3 попытки на прокси, средний пинг
Расчёт стабильности (успешные/всего × 100)
Выбор ТОП-2 по стабильности → пингу
3.2. Новые функции (дополнительные)
3.2.1. Геолокация прокси
Определение страны по IP через встроенную базу GeoLite2 (без внешних API-запросов)
Отображение флага страны в таблице
Фильтр по странам (чекбоксы)
3.2.2. Проверка анонимности
Прозрачный — передаёт реальный IP клиента
Анонимный — скрывает реальный IP, но сообщает о использовании прокси
Элитный (High Anonymous) — полностью скрывает прокси
Отображается в таблице значком: 🔴/🟡/🟢
3.2.3. Проверка скорости скачивания
После успешного пинга: замер скорости скачивания тестового файла (1 МБ)
Результат в Кбит/с, отображается в таблице
Таймаут: 15 секунд
3.2.4. История проверок
SQLite-база history.db рядом с exe
Сохранение всех проверок с датой, результатами, статистикой
Кнопка "История" — открывает окно с фильтрами и графиком успешности по дням
3.2.5. Автоматический режим
Таймер: проверка каждые N минут (настраивается: 15/30/60/120)
При обнаружении новых лучших прокси — уведомление в трее (balloon tip)
Фоновая работа (сворачивание в трей)
3.2.6. Экспорт результатов
Table
Формат	Содержимое
TXT	тип://ip:port (один прокси на строку)
JSON	Полная структура с метаданными
CSV	Таблица для Excel
Буфер обмена	Быстрое копирование
3.2.7. Настройки (окно "⚙ Настройки")
Количество прокси из источника (1–10, по умолчанию 3)
Таймауты (подключение, чтение)
Количество потоков (1–50, по умолчанию 10)
Целевые URL для проверки (свои или предустановленные)
Язык интерфейса (RU/EN)
Тема оформления (Светлая/Тёмная)
Автозапуск проверки при старте программы
4. Технические требования
4.1. Потоки и асинхронность
GUI работает в главном потоке (не блокируется)
Проверка прокси — в пуле потоков (ThreadPoolExecutor / аналог)
Обновление GUI через механизм message queue / invoke
Кнопка "Стоп" — мгновенная отмена всех операций
